# OpenNyAI - Sample Legal Documents

This directory contains raw legal documents for processing.

## Supported Document Types

- **Court Judgments**: Supreme Court and High Court judgments
- **FIRs**: First Information Reports
- **Legal Notices**: Official legal notices and orders
- **Case Diaries**: Investigation case diaries

## Data Organization

Place your documents in the following structure:

```
raw/
├── judgments/       # Court judgment documents
├── firs/            # FIR documents
├── notices/         # Legal notices
└── case_diaries/    # Case diary entries
```

## Data Format

Documents can be in the following formats:
- PDF files (.pdf)
- Text files (.txt)
- Word documents (.docx)
- JSON files (.json)

## Sample Data Sources

- [Indian Kanoon](https://indiankanoon.org/) - Indian legal documents
- [Supreme Court of India](https://main.sci.gov.in/) - Supreme Court judgments
- [ILDC Dataset](https://github.com/Exploration-Lab/ILDC) - Indian Legal Documents Corpus
