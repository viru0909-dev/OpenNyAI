# Notebooks

This directory contains Jupyter notebooks for data exploration, preprocessing, model training, and evaluation.

## Notebooks

1. **01_data_exploration.ipynb** - Explore and understand the legal document dataset
2. **02_preprocessing.ipynb** - Data preprocessing and feature engineering
3. **03_model_training.ipynb** - Model training experiments
4. **04_evaluation.ipynb** - Model evaluation and analysis

## Getting Started

```bash
# Install Jupyter
pip install jupyter jupyterlab

# Start Jupyter Lab
jupyter lab
```
